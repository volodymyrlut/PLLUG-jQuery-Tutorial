# PLLUG-jQuery-Tutorial
##Simple jQuery tutorial for PLLUG web branch community (UA)
###Функції
######В java script функції бувають іменованими або анонімними. Іменовані функції доступні в усій [області видимості](https://ru.wikipedia.org/wiki/%D0%9E%D0%B1%D0%BB%D0%B0%D1%81%D1%82%D1%8C_%D0%B2%D0%B8%D0%B4%D0%B8%D0%BC%D0%BE%D1%81%D1%82%D0%B8), анонімні- тільки після їх оголошення. Найчастіше анонімні функції запускаються самі за допомогою `(function(){}())` структури.
